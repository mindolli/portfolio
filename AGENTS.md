# 포트폴리오 웹사이트

## 내용

### 설명 (코드에 안 들어가는 부분임 진짜 설명)

- 내가 찍은 영화 두 개
- 본문이 있고, 아래에 스틸 컷 세 장을 한 줄에 삽입할 것임
- 스틸컷 이미지를 누르면 크게 볼 수 있음

- 영화는 두 편

### 본문 (실제 코드에 들어가는거)

- 제목
- 촬영년도
- 장르
- 직책
- 시놉시스
- 스틸컷

위 내용이 두 번 들어갈 수 있도록 더미 데이터 넣어줘.

## UI

- 꾸밈 없이
- html, css로만 구현하기
- 본문 폰트: Gothic A1
  - <head>
      <link rel="preconnect" href="https://fonts.googleapis.com">
      <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
      <link href="https://fonts.googleapis.com/css2?family=Gothic+A1&display=swap" rel="stylesheet">
  - CSS class
    .gothic-a1-regular {
    font-family: "Gothic A1", sans-serif;
    font-weight: 400;
    font-style: normal;
    }
